Using GRUJ as a Simple Build Tool launch launcher

The easiest way to see this in action would be to checkout the GRUJ website from either Launchpad or GitHub and simply run the thing locally.

The web project includes:

  • SBT 0.11.0, with included plugins:
    • SBT eclipse 1.4.0
    • XSBT Web plugin 0.2.8
    • Coffeescripted SBT plugin 0.1.5
    • Less SBT plugin 0.1.1
  • Lift web framework 2.4-M4, running on:
    • Jetty 8.0.4.v20111024

Required steps:

Branch the project from Launchpad or download and extract it to some folder. 

user@box:~$ bzr branch lp:gruj
Branched 33 revision(s).
user@box:~$ cd gruj

… alternatively, clone from GitHub: 

user@box:~$ git clone git://github.com/melezov/gruj.git

user@box:~$ cd gruj

Run prepare-workspace.sh (or .bat if on Windoze). This will fire up GRUJ and download SBT 0.11.x launcher from Typesafe repos. Although the launcher download will only take a few seconds, the update process could take a while if you do not have the dependencies in your ivy cache already.
